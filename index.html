Types et inférence de type

En JavaScript, les types de variables principaux sont : integer, string, boolean et array. JavaScript déduit automatiquement le type d'une variable (inférence de type).

Tout est objet

Tout en JavaScript est un objet. Une fois le type détecté, la variable peut utiliser toutes les méthodes associées à ce type.

Programmation orientée objet (POO)

Une classe est une représentation abstraite. Un objet est une instance concrète d'une classe. Exemple : classe Humain, chaque personne est un objet.

Propriétés et méthodes

  • Propriétés : définissent l'état ou les caractéristiques (ex : nom, dateDeNaissance)
  • Méthodes : représentent les actions que l'objet peut effectuer (ex : parler(), marcher())
